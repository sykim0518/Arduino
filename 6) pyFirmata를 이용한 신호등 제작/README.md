pyFirmata를 이용한 신호등 제작
=============
>### 목표

&nbsp;&nbsp; 신호등을 PyFirmata를 이용하여 파이썬으로 구현한다

* 신호등 작성

  - 차량용 직진, 정지준비, 정지 
  
  - 보행자용 신호 

* 보행자용 신호등 점멸 기능 추가
  
  - 보행자용 신호등 유지시간 30초 
  
  - 유지시간 중 마지막 10초에서는 보행자 직진 신호 등 점멸 


* 보행자용 호출 단추 기능 추가

  - 호출 후 5초 뒤에 보행자 신호등 직진으로 변경
